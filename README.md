# 📚 Online Course Material Management System (OCMMS)

## 📌 Description
A web-based application developed using **PHP and MySQL** to manage courses, subjects, classes, and study materials. This project was developed as my **Diploma Final Year Project** and demonstrates practical implementation of full-stack web development concepts such as database design, CRUD operations, authentication, file handling, and dashboard analytics. Administrators can upload and manage academic content, while users can register and download materials in an organized digital format.

---

## 🎯 Project Overview
The system simplifies digital distribution of academic resources by providing a centralized platform where administrators can manage courses and materials efficiently, and students can securely access and download content anytime. It reduces manual work and improves accessibility within an institution.

---

## 🛠️ Technologies Used
- PHP  
- MySQL  
- HTML  
- CSS  
- Bootstrap  
- JavaScript  
- Chart.js  

---

## ✨ Key Features
- Admin Login / Logout  
- Add & Manage Classes, Courses, Subjects  
- Upload PDF Study Materials  
- User Registration & Login  
- Download Course Material 
- Dashboard with Statistics
